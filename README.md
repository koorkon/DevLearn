DevLearn is a full-stack educational tool that uses AI to help students and professionals learn faster. Users can upload documents (PDF/TXT) to automatically generate summaries, multiple-choice questions (MCQs), and flashcards.

**Live Demo:** [https://devlearn-lilac.vercel.app](https://devlearn-lilac.vercel.app)

## ‚ú® Features

* **AI Summary:** Automatically extract key points from long documents.
* **MCQ Generator:** Generate interactive quizzes to test your knowledge.
* **Flashcards:** Create digital flashcards for quick revision.
* **Secure Backend:** Integrated with Groq AI API for high-speed processing.
* **CI/CD Pipeline:** Fully automated testing and deployment via GitHub Actions and Docker.
